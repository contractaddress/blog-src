Hey! welcome to my first blog post, today i'll walk you through using a yubikey with veracrypt to add an extra layer of security to your containers

## How it works
We program the yubikey slot2 with the `ykman` CLI tool to use the challenge-response method and store a 20byte key.

the yubi slot will take a string (challenge) as input -> run `HMAC-SHA1` with the challenge & key -> returns a response we place in a tempory file in ram `(/dev/shm/)`

this becomes the keyfile for the container, so the challenge must always be the same (and does not need to be kept secret). We use it to create/mount containers and delete the file afterwards

so naturally the easiest way to manage all this, is with a script.

## required packages:

```bash
sudo pacman -S yubikey-personalization yubikey-manager veracrypt
```
#
## program slot 2 with challenge response:

```bash
ykman otp chalresp --touch 2
```
<p class='small-text' style={{color: 'var(--muted)'}}>(backup the private key, if you ever lose your yubikey you can reprogram slot2 with it)</p>

Now your slot is ready for use! you can grab my script at the very end of the page or follow the next steps for a manual setup

<h1 style={{
    textDecoration: 'underline dashed',
    color: 'var(--accent)'
}}>Manual Setup</h1>
derive temporary keyfile from the yubikey to ram: (touch when it blinks)
```bash
ykchalresp -2 "my-veracrypt-challenge" > /dev/shm/vc-keyfile
```

## create the container
<p class='small-text'>change to desired encryption, filetype ect</p>
```bash
veracrypt --text --create ~/tutorial_container \
  --size=3G \
  --keyfiles="/dev/shm/vc-keyfile" \
  --encryption=AES-Twofish \ 
  --hash=SHA-512 \
  --filesystem=ext4 \
  --volume-type=normal \
  --random-source=/dev/urandom
```
Add a password! the yubikey should be 2FA not a passkey.

shred the file after making the container
```bash
shred -u /dev/shm/vc-keyfile
```

## Create an mount/unmount script:

`verayubi_mount.sh`

```bash
#!/bin/bash
set -e

CHALLENGE="my-veracrypt-challenge"
CONTAINER="$HOME/tutorial_container" #change to appropriate container name/location
MOUNTPOINT="/run/media/$USER/priv"
KEYFILE=$(mktemp -p /dev/shm)

echo "Touch your YubiKey..."

#Derive keyfile from YubiKey (requires physical touch)
ykchalresp -2 "$CHALLENGE" > "$KEYFILE"

#Mount — will prompt for passphrase
veracrypt --text --keyfiles="$KEYFILE" --protect-hidden=no "$CONTAINER" "$MOUNTPOINT"

#Wipe keyfile from RAM immediately
shred -u "$KEYFILE"

echo "Mounted at $MOUNTPOINT"
```

unmount script:

`verayubi_unmount.sh`

```bash
#!/bin/bash
veracrypt --text -d /mnt/private
echo "Unmounted !"
```

<summary style={{ 
  color: 'var(--accent)', 
  textDecoration: 'underline dashed',
  textUnderlineOffset: '6px',
  marginBottom: '10px'
}}>Here is a script to create/mount/unmount yubi+vera containers</summary>

<p style={{fontSize: '14px'}}>script does not program slot2 with the chalresp token, you need to provide it yourself</p>
```bash
#!/bin/bash
set -e

# Config
CHALLENGE="my-veracrypt-challenge"
MOUNTPOINT="/run/media/$USER/priv"
CONT_DIR=$HOME/.local/share/verayubi

if [ ! -d "$CONT_DIR" ]; then
    mkdir -p $CONT_DIR
fi

create() {
    read -p "Container Name: " CONT_NAME 
    read -p "Size (format: 2K, 2M, 2G): " CONT_SIZE 

    echo "Touch your Yubikey..."

    # Derive keyfile from YubiKey 
    ykchalresp -2 "$CHALLENGE" > /dev/shm/vc-keyfile

    veracrypt --text --create $CONT_DIR/$CONT_NAME \
      --size=$CONT_SIZE \
      --keyfiles="/dev/shm/vc-keyfile" \
      --encryption=AES-Twofish \
      --hash=SHA-512 \
      --filesystem=ext4 \
      --volume-type=normal \
      --random-source=/dev/urandom

    trap 'shred -u /dev/shm/vc-keyfile 2>/dev/null' EXIT
}

mount_vol() {
    select CONTAINER in "$CONT_DIR"/*; do
        break
    done

    MOUNTPOINT="/run/media/$USER/priv"
    KEYFILE=$(mktemp -p /dev/shm)

    if [ ! -d "$MOUNTPOINT" ]; then
        sudo mkdir -p "$MOUNTPOINT"
    fi

    echo "Touch your YubiKey..."

    # Derive keyfile from YubiKey
    ykchalresp -2 "$CHALLENGE" > "$KEYFILE"

    veracrypt --text --keyfiles="$KEYFILE" --protect-hidden=no "$CONTAINER" "$MOUNTPOINT"

    trap 'shred -u "$KEYFILE" 2>/dev/null' EXIT

    echo "Mounted at $MOUNTPOINT"
    #nautilus -w $MOUNTPOINT or whatever file manager u use
}

unmount_vol() {
    veracrypt --text -d $MOUNTPOINT
    echo "Unmounted"
}

list() {
    ls $CONT_DIR
}


case "$1" in
    create)   create ;;
    mount)    mount_vol ;;
    unmount)  unmount_vol ;;
    list)     list ;;
    *)        echo "Usage: verayubi {create|mount|unmount|list}" ;;
esac
```
